HTML

  • the head element contains information about the webpage
  • the body element represents content that will be visible to the viewer
  • img elements can and should have alt tags for page optimization
  • list items such as these may be numbered or un-numbered
  • p elements contain paragraphs of texts

CSS

  • CSS is a style sheet associated to the website via link in the html header
  • css properties consist of selectors (h1, p) properties(color, height) and values of specific items
  • Each element is essentially a box containing a margin, border, padding, and the content within the element itself

Git

  • Connecting your files to the Git is ensures that your work is being saved
  • Ensuring that you are working on the correct branch eliminates version issues
  • Git add "file name here" will add the files that are currently not being tracked to the branch
  • Git Push transmits your local branch to the repository on the hub
